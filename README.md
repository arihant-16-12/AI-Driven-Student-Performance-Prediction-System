# 🎓 AI-Driven Student Performance Prediction System

## 📌 Project Overview
This project predicts a student's academic performance using Machine Learning and provides AI-based study recommendations.

## ✨ Features
- Student Performance Prediction
- Personalized Study Plan
- Weakness Analysis
- Improvement Tips
- Weekly Study Schedule
- Performance Visualization

## 🛠 Technologies Used
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Matplotlib

## 📊 Input Features
- Attendance
- Assignment Score
- Quiz Score
- Midterm Score
- Study Hours
- GPA

## 🚀 How to Run

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Run the application:

```bash
streamlit run app.py
```
